# jarvisoj.com -- CRYPTO -- 影之密码

## Challenge

```
请分析下列密文进行解密 8842101220480224404014224202480122 得到flag，flag为8位大写字母

题目来源：CFF2016
```

## Solution

严格意义上讲，这不能叫做密码，充其量只能算一种编码方案。

解码方案如下：

1. 字符 `'0'` 为分隔符。

2. 分隔后，每个单元是由 `'1'`、`'2'`、`'4'`、`'8'` 构成的字符串。

   把每个单元的数字字符当成十进制数字相加。

3. 每个单元数字相加的结果翻译成第n个英文字母，最后就可以的到flag。
