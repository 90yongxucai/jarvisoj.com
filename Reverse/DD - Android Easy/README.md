## Challenge

```
Flag 为下一关邮箱。

```

## Solution

1. 下载，zip解压。

2. 用 `d2j-dex2jar.bat` 将 `classes.dex` 转化成 `classes-dex2jar.jar`。

3. 用 `jd-gui` 打开 `classes-dex2jar.jar`，很容易看到 `com.didi_ctf.flagapp`。进去，虽然去除了符号信息，然而我们只要复制相关代码、编译就可以看到结果。
